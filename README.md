# Open Worlds - UGC Platform Website

Welcome to the Open Worlds website repository! This is the official website for the Open Worlds User-Generated Content (UGC) platform.

## 🌐 Website Overview

Our website features:
- **Home Page**: Introduction to Open Worlds with key features
- **About Section**: Platform information and core features
- **Careers Page**: Job opportunities and benefits
- **Community Page**: Social links and community guidelines
- **Contact/Support**: Get in touch with our team

## 🎨 Design

- **Color Scheme**: Blue (#0066cc) as primary, white as secondary
- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Modern UI**: Clean, professional interface

## 📁 File Structure

```
.
├── index.html          # Main home page
├── careers.html        # Careers page
├── community.html      # Community page
├── styles.css          # Shared styles
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## 🚀 Getting Started

1. The website is automatically hosted on GitHub Pages at: `https://openworldssupport.github.io`

2. To make changes:
   - Clone this repository
   - Edit the HTML, CSS, or JavaScript files
   - Commit and push your changes
   - The site will automatically update

## ✨ Features

- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Contact Form**: Functional contact form with validation
- **Responsive Layout**: Mobile-first design
- **Social Media Integration**: Links to Discord, Twitter, YouTube, and Instagram
- **Animations**: Smooth animations and transitions throughout

## 🔧 Customization

### Update Contact Information
Edit the contact section in `index.html`:
- Email: `support@openworlds.com`
- Phone: `+1 (555) 123-4567`
- Address: Update with your actual address

### Update Social Links
Replace the `#` in social media links with your actual profiles:
- Discord: `https://discord.gg/yourserver`
- Twitter: `https://twitter.com/yourhandle`
- YouTube: `https://youtube.com/yourhandle`
- Instagram: `https://instagram.com/yourhandle`

### Update Job Listings
Edit `careers.html` to add, remove, or modify job postings.

### Update Community Events
Edit `community.html` to add or update upcoming events.
